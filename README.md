# FIFA Threat Shield ⚽🛡️

An AI-powered phishing detection system that helps identify fake FIFA ticketing and merchandise websites using multiple cybersecurity signals.

## Features

- WHOIS Domain Intelligence
- SSL Certificate Validation
- Visual Similarity Detection
- OSINT Mention Analysis
- AI-based Risk Scoring
- Interactive Dashboard

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- FastAPI
- Python
- Pandas

## Project Structure

```
Frontend/
Backend/
```

## Running the project

### Backend

```bash
cd Backend
pip install -r requirements.txt
python -m uvicorn main:app --reload --port 8000
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
